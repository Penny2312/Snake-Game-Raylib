# Snake Game using Raylib

A simple snake game using Raylib Library in C++ from Programming with Nick. Raylib is an open source library in which alot is possible. In order to use this code you will need to install Raylib and put this repo in this folder:<br>
`"C:\raylib\projects\snake"`

Please check out this link for the Raylib Repo:<br>
https://github.com/raysan5/raylib

Please check use this Snake template file in which you can create your Raylib projects easily with everything set up, all you need to do is wr hereite your C++ code in `\game\src`. From just pres F5 and your code will start running and Raylib will open the window.
